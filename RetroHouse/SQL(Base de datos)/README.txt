- Para poder crear correctamente la base de datos, primero debe ejecutar el archivo "RetroHouseBD"
ya que este contiene todas las tablas necesarias para la base de datos.

- Para llenar la base de datos, se debe ejecutar el archivo "LlenarRetroHouseDB", ya que este contiene
toda la información basica necesaria para que el aplicativo funcione.

Admin123.

solo se debe agregar en los siguientes archivos:

RetroHouse/Database/databaseConn.php y RetroHouse/Database/databaseUtils.php en la sexta linea de código 
cada uno.
